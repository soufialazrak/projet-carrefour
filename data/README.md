# Données du projet

Ce dossier contient les données utilisées dans le projet.

- `archive/` : dataset source (Olist e-commerce)
- `outputs/` : fichiers CSV générés après transformation

Les fichiers CSV dans `outputs/` sont utilisés comme source d’ingestion pour alimenter la base PostgreSQL.

Ces données constituent une couche proche d’un Data Lake (données brutes avant structuration).
